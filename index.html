📌 Introdução

Este projeto usa o GitHub Pages para hospedar um site estático de forma gratuita. O repositório pdrfranca/pdrfranca.github.io é automaticamente publicado como um site pessoal.

"Sites feitos com propósito contam histórias. Este conta a minha." — Pedro França

📁 Estrutura do Repositório

Mostrar detalhes da estrutura
  • index.html — Página principal
  • /assets — Imagens, logos, mascotes
  • CNAME — (opcional) domínio personalizado

🛠️ Como o Repositório Foi Criado

  1. Acesse: github.com/new
  2. Digite o nome: pdrfranca.github.io
  3. Marque como público
  4. Crie o repositório
  5. Adicione seu index.html e envie via git ou editor online

🚀 Publicação no GitHub Pages

O GitHub detecta automaticamente o repositório username.github.io e publica em:

🔗 https://pdrfranca.github.io

Não é necessário configurar manualmente os Pages nesse caso.
